# Model of self-driven particles

The one bacterium model code file is for modelling only one rod-like shaped bacterium, which can be simplified by theree sphere connected by two springs. For only one bacterium, there are the self-drive force, random force, spring force and bending force of the spring acting on the bacterium. Once we calculated the force acting on those three spheres or the bacterium, we can use the Langevin Leap-frog method of integration to calculate the displacement for each time step.

Three bacteria model code file consists of three bacteria, the van-der-waals interaction was included in this model.
